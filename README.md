# 🚀 System Monitor API

A lightweight FastAPI-based monitoring tool to track Linux server metrics in real-time.

## 🛠 Tech Stack
- **FastAPI**
- **Psutil**
- **Pydantic**
- **Uvicorn**

## 🚀 Quick Start
```bash
# Clone the project
git clone <your-repo-link>
cd FastAPI

# Install dependencies
pip install -r requirements.txt

# Run the server
python3 -m uvicorn app.main:app --reload

   Features

    Real-time system hardware tracking.

    JSON-formatted professional responses.

    Built with modular architecture (Routers & Schemas).
